# NutriBot
---

## Overview
This app is designed to provide nutrition details based on food photos and text input. It uses AI algorithms to analyze food images and extract nutritional information.


## Features
- **Image Recognition:** Identifies food items from photos.
- **Nutritional Analysis:** Provides detailed nutrition information.
- **Text Input:** Allows users to input food names for analysis.
- **User-Friendly Interface:** Simple and intuitive design.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/food-nutrition-app.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
1. Run the application:
   ```bash
   python app.py
   ```
2. Upload a food photo or enter the food name in the text input field.
3. View detailed nutrition information about the food item.
